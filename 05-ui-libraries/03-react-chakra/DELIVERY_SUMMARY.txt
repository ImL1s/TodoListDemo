📊 與其他 UI 庫對比總結
───────────────────────────────────────────────────────────────────────────────

Chakra UI vs Material-UI (MUI):
  • 包體積小 50% (~40KB vs ~80KB)
  • API 更簡單直觀
  • 主題定製更容易
  • 學習曲線更低

Chakra UI vs Ant Design:
  • 包體積小 75% (~40KB vs ~150KB+)
  • 設計更靈活
  • 定製更簡單
  • Style Props 系統

Chakra UI vs Tailwind CSS:
  • 組件化 vs 工具類
  • 內置豐富 Hooks
  • TypeScript 支持更好
  • 開發效率更高
