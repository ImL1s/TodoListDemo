快速命令
================================================================================

开发:
  npm install          # 安装依赖
  npm run electron:dev # 启动开发环境
  npm run type-check   # TypeScript 类型检查

构建:
  npm run build        # 构建当前平台
  npm run build:win    # 构建 Windows
  npm run build:mac    # 构建 macOS
  npm run build:linux  # 构建 Linux

工具:
  npm run preview      # 预览构建结果
  npm outdated         # 检查过期依赖
  npm audit            # 安全审计

================================================================================
核心文件说明
================================================================================

electron/main.ts
  主进程入口文件，负责：
  - 创建和管理应用窗口
  - 定义 IPC 处理器（getTodos, addTodo, etc.）
  - 实现文件系统操作（读写 todos.json）
  - 创建原生应用菜单
  - 管理系统托盘
  - 处理应用生命周期

electron/preload.ts
  Preload 脚本，负责：
  - 使用 contextBridge 暴露安全的 API
  - 封装 IPC 通信方法
  - 定义 TypeScript 类型
  - 提供事件监听接口

src/App.vue
  Vue 根组件，负责：
  - 管理全局状态（todos, filter）
  - 调用 electronAPI 进行 IPC 通信
  - 实现业务逻辑（添加、编辑、删除待办）
  - 处理数据导入导出
  - 显示统计信息和模态对话框

src/components/TodoInput.vue
  输入组件，负责：
  - 提供输入框和添加按钮
  - 处理用户输入
  - 显示字符计数
  - 暴露 focus() 方法

src/components/TodoList.vue
  列表组件，负责：
  - 渲染待办列表
  - 管理列表动画
  - 转发事件到父组件

src/components/TodoItem.vue
  待办项组件，负责：
  - 显示单个待办事项
  - 实现双击编辑功能
  - 提供完成/删除操作
  - 显示创建时间

vite.config.ts
  Vite 配置文件，负责：
  - 配置 Vue 插件
  - 配置 Electron 插件
  - 设置路径别名
  - 配置开发服务器
  - 优化构建输出

electron-builder.json
  打包配置文件，负责：
  - 定义应用 ID 和名称
  - 配置各平台打包选项
  - 设置输出目录
  - 配置更新服务器

================================================================================
开发工作流
================================================================================

1. 开发阶段:
   npm install          → 安装依赖
   npm run electron:dev → 启动开发环境
   编辑代码              → HMR 自动更新
   npm run type-check   → 检查类型

2. 测试阶段:
   npm run preview      → 预览构建
   手动测试功能          → 验证功能正常

3. 构建阶段:
   npm run build:win    → 构建 Windows
   npm run build:mac    → 构建 macOS
   npm run build:linux  → 构建 Linux

4. 发布阶段:
   创建 GitHub Release  → 上传构建产物
   更新 CHANGELOG       → 记录变更
   发布到各平台商店      → 分发应用
