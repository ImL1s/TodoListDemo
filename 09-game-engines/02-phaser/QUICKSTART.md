# Phaser Todo List - 快速入門

## 5 分鐘快速體驗

### 1. 安裝依賴

```bash
npm install
```

### 2. 啟動開發服務器

```bash
npm run dev
```

瀏覽器會自動打開 `http://localhost:3000`

### 3. 開始使用

#### 添加任務
1. 在頂部輸入框輸入任務
2. 選擇優先級（低/中/高）
3. 點擊 "Add" 或按 Enter

#### 完成任務
- 點擊任務左側的複選框
- 會觸發粒子效果和積分獎勵

#### 刪除任務
- 懸停在任務上
- 點擊右側出現的 ×

#### 拖放排序
- 拖動任務到新位置
- 釋放後自動排序

#### 過濾任務
- All: 顯示所有任務
- Active: 只顯示未完成
- Completed: 只顯示已完成

### 鍵盤快捷鍵

| 按鍵 | 功能 |
|------|------|
| `N` | 聚焦輸入框（新增任務） |
| `C` | 清除已完成的任務 |
| `1` | 顯示全部任務 |
| `2` | 顯示進行中任務 |
| `3` | 顯示已完成任務 |

### 遊戲化功能

#### 積分系統
- 完成任務: +10 分
- 連續獎勵: 每 5 天連續 +5 分
- 每 100 分升一級

#### 連續天數
- 每天完成至少一個任務
- 連續天數會累積
- 中斷會重置為 1

### 構建生產版本

```bash
npm run build
```

構建產物在 `dist/` 目錄，可以直接部署到任何靜態服務器。

### 部署到 Netlify

```bash
# 拖放 dist 目錄到 Netlify Drop
# 或使用 CLI
npm install -g netlify-cli
netlify deploy --prod --dir=dist
```

## 核心技術

- **Phaser 3.70.0**: HTML5 遊戲引擎
- **TypeScript 5.3**: 類型安全
- **Vite 5.0**: 快速構建工具
- **LocalStorage**: 數據持久化
